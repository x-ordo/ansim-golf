# 안심골프 프로젝트 로드맵 (Roadmap)

## Phase 1: MVP 개발 및 검증 (Current)
**목표**: 핵심 가치(에스크로, AI, SaaS)가 동작하는 웹 프로토타입 완성 및 데모.

- [x] **프로젝트 셋업**: React + Vite + Tailwind 환경 구축.
- [x] **기본 UI 구현**: 메인 보드, 부킹 카드, SaaS 대시보드 UI.
- [x] **목 데이터(Mock Data) 구성**: 티타임, 골프장, 매니저 데이터 구조화.
- [ ] **AI 챗봇 연동**: Gemini API 실제 연동 및 프롬프트 최적화.
- [ ] **결제 모듈 샌드박스 연동**: Toss Payments 결제 흐름 완결 (승인/실패 처리).
- [ ] **배포**: Cloudflare Pages 통한 웹 배포.

## Phase 2: 데이터베이스 및 백엔드 고도화 (Next)
**목표**: 실제 데이터 저장 및 상태 관리가 가능한 정식 서비스 구조 마련.

- [ ] **DB 구축**: Cloudflare D1 또는 Supabase 도입.
- [ ] **API 개발**: Mock 데이터를 실제 API 응답으로 교체.
- [ ] **인증 시스템**: 사용자/매니저 회원가입 및 로그인 (OAuth, SMS 인증).
- [ ] **SaaS 기능 강화**: 텍스트 파싱 로직 고도화, 알림톡 연동(BizMessage).

## Phase 3: 모바일 앱 확장 및 마케팅
**목표**: 접근성 강화 및 유저 확보.

- [ ] **PWA / App**: 모바일 최적화 및 앱 패키징 (React Native 고려).
- [ ] **위치 기반 서비스**: 내 주변 골프장 찾기, 지도 뷰.
- [ ] **마케팅**: 초기 파트너 매니저 모집, 베타 테스터 운영.

## Phase 4: 생태계 확장
**목표**: 골프 라이프스타일 슈퍼앱으로 도약.

- [ ] **ERP 연동**: 골프장 내부 시스템과 실시간 재고 연동.
- [ ] **커뮤니티**: 라운딩 후기, 조인 멤버 평판 시스템.
- [ ] **부가 서비스**: 골프장 맛집 추천, 투어 패키지 등 상품 다양화.
