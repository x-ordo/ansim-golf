# 안심골프 (Ansim Golf) - 차세대 골프 부킹 플랫폼 PRD

## 1. 프로젝트 개요
**안심골프(Ansim Golf)**는 기존 골프 부킹 시장의 '노쇼(No-Show)'와 '먹튀' 문제를 해결하고, 매니저와 골퍼 모두에게 투명하고 편리한 경험을 제공하는 **AI 기반 에스크로 골프 부킹 플랫폼**입니다.

### 1.1. 배경 및 목적
- **시장 문제점**:
    - 폐쇄적인 부킹 시장과 불투명한 가격 구조.
    - 선입금 후 매니저 잠적(먹튀) 또는 골퍼의 일방적 노쇼로 인한 금전적 피해 빈번.
    - 비효율적인 수동 예약 관리(카카오톡, 문자 의존).
- **해결 방안**:
    - **에스크로(Escrow) 결제**: 라운딩 완료 시까지 결제 대금을 안전하게 예치하여 '먹튀' 원천 차단.
    - **노쇼 방지 시스템**: 빌링키 기반 위약금 자동 청구 시스템 도입.
    - **AI 매니저**: 자연어 대화를 통한 스마트한 티타임 추천 및 예약 대행.
    - **매니저 SaaS**: 골프장 및 전문 매니저를 위한 통합 예약/회원/매출 관리 솔루션.

### 1.2. 핵심 가치 (Value Proposition)
- **Trust (신뢰)**: 에스크로 결제와 검증된 매니저 시스템으로 안전한 거래 보장.
- **Convenience (편의)**: AI 챗봇을 통한 간편한 검색 및 예약, 직관적인 UX.
- **Efficiency (효율)**: SaaS를 통한 매니저 업무 자동화 (파싱, 정산, 알림).

## 2. 타겟 유저
### 2.1. 일반 골퍼 (Golfer)
- **특징**: 안전한 조인/양도 티타임을 원함. 복잡한 예약 절차를 싫어함.
- **니즈**: 먹튀 걱정 없는 선입금, AI 추천, 투명한 위약금 정책.

### 2.2. 파트너 매니저 (Manager)
- **특징**: 골프장 티타임을 대량 보유하거나 유통하는 전문 매니저.
- **니즈**: 노쇼 리스크 제거, 예약 관리 업무 효율화, 악성 재고(임박 타임) 처리.

## 3. 핵심 기능 (Key Features)

### 3.1. 사용자(골퍼) 기능
1.  **실시간 티타임 예약 (Real-time Booking)**
    - 지역별, 날짜별, 골프장별 실시간 티타임 검색 및 조회.
    - 카드형(BookingCard) 및 리스트형(Board) 뷰 제공.
2.  **안심 결제 (Safe Payment)**
    - Toss Payments 연동 에스크로 결제.
    - 라운딩 종료 후 매니저에게 정산되는 안전 장치.
3.  **AI 챗봇 (AI Concierge)**
    - Gemini API 기반 자연어 티타임 추천 및 상담.
    - "내일 오전 8시 파주 근처 조인 찾아줘" 등의 질의 처리.
4.  **안심 조인 (Safe Join)**
    - 인증된 회원 간의 조인 매칭.

### 3.2. 매니저(SaaS) 기능
1.  **대시보드 (Dashboard)**
    - 예약률, 매출, 노쇼 현황 등 주요 지표 실시간 시각화.
2.  **인벤토리 관리 (Inventory Management)**
    - 텍스트(카톡/엑셀) 파싱을 통한 티타임 자동 등록.
    - 실시간 재고(티타임) 상태 관리 (판매중, 입금대기, 확정 등).
3.  **덤핑 제어 엔진 (Dumping Engine)**
    - 임박한 티타임의 가격을 자동으로 조정하여 판매율 제고.
4.  **노쇼 관리 (No-Show Management)**
    - 예약금/위약금 규정 설정 및 자동 청구.

## 4. 성공 지표 (KPI)
- **거래액 (GMV)**: 에스크로 결제를 통한 월간 총 거래액.
- **부도율 (Default Rate)**: 노쇼 및 먹튀 발생 건수 0건 목표.
- **재사용률 (Retention)**: 서비스 이용 후 재예약 비율.
- **매니저 업무 시간 단축**: 기존 수동 관리 대비 50% 이상 절감.

## 5. 향후 계획
- **Phase 1 (MVP)**: 웹 기반 예약/결제, 기초 SaaS 기능, AI 챗봇 베타.
- **Phase 2 (Mobile App)**: 푸시 알림, 위치 기반 서비스 강화.
- **Phase 3 (Ecosystem)**: 골프장 ERP 직접 연동, 투어/레슨 상품 확장.
